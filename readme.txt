------- Portuguese version ------- 

Existem duas formas para baixar o Software

Primeira (forma fácil):

Apenas baixe o arquivo "Fatigue Life Calculator (FLC).rar"
e cole ele em uma pasta de sua preferência, iniciando
o aplicativo pelo arquivo "FLC_executable.exe"

Secunda (forma trabalhosa):

Baixe o arquivo da pasta "Instalador" e o instale normalmente,
em seguida baixe a pasta "icons" do repositório e cole essa pasta,
dentro da pasta "share" de onde foi instlado.
Isso fará que os icones não fiquem faltando no app.

------- English version ------- 

There are two ways to download the Software

First (easy way):

Just download the "Fatigue Life Calculator (FLC).rar" file
and paste it in a folder of your choice, starting
the application by the file "FLC_executable.exe"

Second (medium way):

Download the file from the "Installer" folder and install it normally,
then download the "icons" folder from the repository and paste this folder,
inside the "share" folder where it was installed.
This will ensure that the icons are not missing from the app.
